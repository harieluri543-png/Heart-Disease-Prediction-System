About This Project

The Heart Disease Prediction System is a Machine Learning-based web application developed to predict whether a person is at risk of heart disease using various medical parameters. The system provides instant predictions, confidence scores, health recommendations, prediction history, and downloadable PDF reports through a simple and user-friendly interface.

Project Objectives

  • ✔ Heart Disease Risk Prediction
  • ✔ Confidence Score Display
  • ✔ Personalized Health Recommendations
  • ✔ Prediction History
  • ✔ PDF Report Generation
  • ✔ Responsive and User-Friendly Interface

Technologies Used

  • Python
  • Flask
  • HTML5
  • CSS3
  • Machine Learning (Scikit-learn)
  • NumPy
  • Pandas
  • Joblib
  • ReportLab (PDF Generation)

Machine Learning Model

This project uses the Random Forest Classifier to predict the possibility of heart disease based on patient medical data. The model is trained using supervised machine learning techniques to provide fast, reliable, and accurate predictions.

Dataset Information

The model is trained using the Heart Disease Dataset, which contains important patient medical records such as age, gender, chest pain type, resting blood pressure, cholesterol level, fasting blood sugar, resting ECG results, maximum heart rate, exercise-induced angina, ST depression (Oldpeak), slope of the ST segment, number of major vessels, and thalassemia. These features are used to determine the likelihood of heart disease.
